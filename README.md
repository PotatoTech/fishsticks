# fishsticks

[![Crates.io][crates-io-badge]][crates-io]
[![Documentation][docs-badge]][docs]
[![License][license-badge]](LICENSE)

Wrapper library for handling gamepad input. Currently supports the following backends:

- SDL2
- gilrs
